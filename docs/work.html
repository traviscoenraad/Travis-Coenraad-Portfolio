Programming Projects

Security Log Analyser

Technologies Used: Python, argparse, dataclasses, datetime, unittest

A command-line tool that parses authentication log files and automatically detects security threats including brute force attacks, persistent suspicious IPs, and abnormal login frequency using rule-based detection with sliding window algorithms. Built with a modular architecture separating ingestion, parsing, detection and reporting concerns, demonstrating practical cybersecurity knowledge and professional software engineering practices including Git feature branching and unit testing.

What I'd improve: I would add a web interface allowing users to upload log files through a browser rather than the command line which makes the tool accessible to non-technical security staff.
