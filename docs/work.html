Programming Projects

Bus Ticket Selection and Purchase System (Python Coursework)

Description: This project is a menu-driven Python applciation that allows the user to view, select and purchase bus ticket top-ups. This is done by reading the data from a CSV file. The project focuses on applying obeject-oriented programming and structured file handling.

Technologies Used: Git, Python

Links: GitHub URL | https://olympus.ntu.ac.uk/N1405261/BusTicketPythonProject

Explanation: This project is important because it demonstrates my first experience building a complete software system. It helped me develop to better understand problem solving and system design which is essential for a future Solutions Architect. Being able to understand how application logic, data handling and modular design supports my goal to design scalable systems.

Context: This project was developed as part of my university coursework at Nottingham Trent University. I worked independently to design this system and I applied concepts introduced to me during lectures and practical sessions. This project involved planning the program structure, creating object-oriented classes and implementing file handling using CSV data.
